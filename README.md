# 抽象话生成器

> 躲病毒在家实在是百无聊赖也不知道干什么,元宵节大家说`抽象话`猜灯谜,突然发现我不够抽象,于是心生一计,写下着个抽象的生成器

原理很简单各位看看就懂

特别要感谢**chenxuuu**提供的词库他有一个[网页版](https://cxh.papapoi.com)

[他的项目地址](https://github.com/chenxuuu/chouxianghua)

**主要的功能有:**

- 汉字转换抽象话
- 一键复制
- 快捷发送

**将会有的功能:**

- 自定义词组
- 在线更新词库
- 导出和导入词库
- 抽象转拼音
